## 📌 专升本C语言核心考点（按重要度排序）
### 1. 基础语法与数据类型（送分题）
- 必考点：int/char/float 的区别、变量命名规则、运算符优先级

### 2. 流程控制（if/switch/循环）
- 高频题型：闰年判断、求素数、阶乘计算

### 3. 数组与字符串（易错重灾区）
- 必考点：数组下标从 0 开始、字符串结束符 `\0`、常用字符串函数（strlen/strcmp）
- 题型：注意区分整型数组和字符串数组。

### 4. 函数（专升本重点）
- 核心难点：值传递 vs 地址传递、递归函数
- 常考代码：冒泡排序、求最大公约数、阶乘递归实现

### 5. 指针（很多人卡在这里）
- 必考点：指针与数组的关系、指针变量的定义
- 避坑提醒：搞清楚指针的本质，明确指针存放的是什么。

## ⏰ 我的备考时间规划
- **基础阶段（第1-2个月）**：跟着视频过一遍语法，吃透教材例题
- **强化阶段（第3个月）**：分模块刷题，重点攻克薄弱项（核心是走出刷题舒适圈），并回归课本
- **冲刺阶段（考前1个月）**：对知识进行全面复盘
